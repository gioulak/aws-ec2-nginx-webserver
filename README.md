# aws-ec2-nginx-webserver

## Project Overview
Deployed a static website on AWS EC2 using Nginx and secured the server with firewall and SSH practices.

## Technologies
- AWS EC2
- Ubuntu Linix
- Nginx
- UFW Firewall
- SSH

## Architecture
User → Internet → AWS EC2 → Nginx → Website

## Skills Demonstrated
- Cloud provisioning
- Linix server administration
- Web server configuration
- Security hardening
